Overview

This project is an Android application built using Kotlin and Jetpack Compose that retrieves and displays real-time weather data using the OpenWeather API. The application allows users to enter a city name and view current temperature, weather conditions, and basic forecast information.

Features

Real-time weather data retrieval
City-based weather search
Temperature and weather condition display
Basic forecast display
Error handling for invalid input
Loading state handling

How It Works

User enters a city name.
Application sends a request to the OpenWeather API.
API returns weather data in JSON format.
Response is parsed into Kotlin data classes.
UI is updated using Jetpack Compose state management.
Temperature, condition, and forecast are displayed on screen.

Technologies Used

Kotlin,
Jetpack Compose,
OpenWeather API,
Retrofit (for API calls),
MVVM Architecture.

Future Improvements
